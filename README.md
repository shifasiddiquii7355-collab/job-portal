# 💼 Job Portal

A modern and responsive Job Portal built with React and Tailwind CSS. Users can search, sort, save, and explore remote job opportunities with a clean and intuitive interface.

## 🚀 Live Demo

https://job-portal-shifa.netlify.app/

## ✨ Features

- Search jobs by title or company
- View detailed job information
- Save and remove favorite jobs
- Sort jobs by title, company, or newest
- Responsive design for desktop and mobile
- Loading and empty states

## 🛠️ Technologies Used

- React
- React Router
- Tailwind CSS
- JavaScript (ES6+)
- Vite
- Remotive Jobs API

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/shifasiddiquii7355-collab/job-portal.git
```

Navigate to the project folder:

```bash
cd job-portal
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
